# Jinhee_PF
React와 javascript로 개발된 저를 소개하는 오진희의 main_portfolio 웹사이트 입니다.
PC와 모바일에서 보기 적합하도록 반응형 웹사이트로 제작하였습니다.
저를 소개하는 포토폴리오 인만큼 좀더 근사하고, 다양한 animation과 인터렉티브한 웹페이지로 
제작하기 위해 CSS와 Javascript로 직접 개발하였습니다.
